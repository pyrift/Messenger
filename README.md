# 💬 Messenger

A real-time web messaging app built with Django.

🔗 **Live Demo:** [messenger-2-20ja.onrender.com](https://messenger-2-20ja.onrender.com)

## Features

- User registration and login
- Private chat between users
- Real-time messages (polling every 3 seconds)
- Image sharing in chats
- User profiles with avatars
- Edit and delete messages
- Dark themed UI

## Tech Stack

- **Backend:** Django 6, Python 3.12
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite
- **Deployment:** Render

## Installation

```bash
git clone https://github.com/pyrift/Messenger.git
cd Messenger
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
